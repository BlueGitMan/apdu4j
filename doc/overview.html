apdu4j provides various utilities for working with real life smart cards and readers, with all the glitches and corner cases.

Source code is available on Github: https://github.com/martinpaljak/apdu4j

Useful utilities

  • {@link apdu4j.LoggingCardTerminal} - wrap this around any CardTerminal to get a useful APDU log
  • {@link apdu4j.SCTool} - command line utility, with more documentation on Github
  • {@link apdu4j.TerminalManager} - workarounds for broken PC/SC detection, alternative providers etc
  • {@link apdu4j.json.RemoteTerminal} - implementation of the remote EMV terminal protocol
  • {@link apdu4j.PinPadTerminal} - implementation of CCID pinpads
